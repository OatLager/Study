C++
- 자료형
  - void : 반환 값이 없는 경우 사용. 
  
- 변수 & 메모리 종류

  (변수)
  1. 지역변수
     - 메모리 스택 영역
     - 해당 함수/구역에서만 사용 가능한 변수
  2. 전역변수
     - 메모리 데이터 영역
     - 해당 파일 내에서만 사용 가능한 변수
  3. 정적변수(static)
     - 메모리 데이터 영역
  4. 외부변수(exturn)
     - 메모리 데이터 영역

  (메모리)
  1. Stack 영역
     - 함수 시작 시 생성 / 종료 시 해제
  2. Data 영역
     - 프로그램 시작 시 생성 / 종료 시 해제
  3. 읽기 전용(코드, ROM)
  4. 힙 영

  * Stack : 선입후출, 후입선출 구조
  * Queue : 선입선출, 후입후출 구조 
  
- 함수
  : Stack 구조 

  * 재귀함수 : 함수 내에서 자기 자신을 호출하는 함수.
               가독성, 구현의 용이. 그러나 성능이 떨어짐(느리다) 
               
- 분할구현 
   : Header(.h)와 cpp(.cpp) 파일을 분리하여 생성. 
   Header : 함수/변수의 선언.
   cpp : 함수의 정의/구현. 
   
   - 분할구현의 문제
